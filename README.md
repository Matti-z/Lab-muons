## Overview
Minimal starter README for a research/experiment repo focusing on muon detector data collection, simulation, and analysis tools.

## Requirements
- Python 3.8+
- pip
- Optional: conda, C++ compiler (for native extensions), Jupyter

## Installation
Clone and create a virtual environment:
```bash
git clone <repo-url> .
python -m venv .venv
source .venv/bin/activate     # macOS/Linux
.venv\Scripts\activate        # Windows
pip install -r requirements.txt
```

If using conda:
```bash
conda create -n lab-muons python=3.10
conda activate lab-muons
pip install -r requirements.txt
```
